# Keyway CLI

> One link to all your secrets

## Installation

```bash
npm install -g @keywaysh/cli
```

## Usage
```bash
# Initialize in your project
keyway init

# Pull secrets from vault
keyway pull

# List all secrets
keyway list

# Add a new secret
keyway add API_KEY sk_test_123
```

## Development
```bash
# Install dependencies
npm install

# Run in dev mode
npm run dev

# Build
npm run build

# Test locally
npm link
```
